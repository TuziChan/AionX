# AionUi Tauri 重构 - 规划文档索引

## 项目概述

将 AionUi（基于 Electron 的 AI 代理协作平台）迁移至 Tauri 2.x，后端使用 Rust 完全重写，前端在保持界面和功能完全一致的前提下进行架构优化。
