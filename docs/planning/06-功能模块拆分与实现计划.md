## 3. 模块间接口契约

### 3.1 Agent ↔ Conversation

```rust
// Agent 完成消息后，通过 ConversationService 持久化
impl AgentService {
    async fn on_agent_message(&self, conv_id: &str, event: AgentEvent) {
        match event {
            AgentEvent::MessageComplete { message } => {
                self.conversation_service.add_message(conv_id, message).await;
            }
            // ...
        }
    }
}
```

### 3.2 Cron ↔ Agent

```rust
// Cron 任务触发时，创建临时对话并调用 Agent
impl CronService {
    async fn execute_job(&self, job: &CronJob) {
        let conv = self.conversation_service.create(CreateConversation {
            name: format!("Cron: {}", job.name),
            agent_type: job.agent_type.clone(),
            ..Default::default()
        }).await?;

        self.agent_service.send_message(
            &conv.id,
            &job.prompt,
            None,
            self.event_tx.clone(),
        ).await?;
    }
}
```

### 3.3 Channel ↔ Agent

```rust
// 通道消息到达时，路由到对应的 Agent
impl ChannelService {
    async fn on_channel_message(&self, session: &ChannelSession, content: &str) {
        let conv_id = session.conversation_id.as_ref()
            .ok_or(AppError::NotFound("No linked conversation"))?;

        self.agent_service.send_message(
            conv_id,
            content,
            None,
            self.event_tx.clone(),
        ).await?;
    }
}
```

### 3.4 WebUI ↔ Core Services

```rust
// WebUI 路由直接调用核心服务
async fn api_send_message(
    State(state): State<Arc<AppState>>,
    Json(input): Json<SendMessageInput>,
) -> Result<Json<()>, AppError> {
    state.agent_service.send_message(
        &input.conversation_id,
        &input.content,
        input.files,
        state.event_bus.subscribe(),
    ).await?;
    Ok(Json(()))
}
```
